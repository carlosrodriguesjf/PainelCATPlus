PAINEL CAT PLUS
Este é um Dashboard que exibe o resultado das métricas da Central de Atendimento de uma Universidade ao longo do tempo, permitindo ao usuário filtrar os dados pela data, funcionário, demanda ou período. 
O Dashboard foi desenvolvido utilizando Plotly e Dash
